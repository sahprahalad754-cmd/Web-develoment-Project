🔧 System Information

Technical Implementation

Frontend: Pure HTML5, CSS3, JavaScript (No frameworks)

Styling: Custom CSS with CSS Variables, Glassmorphism design

Animation: Canvas API for particle background, CSS keyframes

Data Storage: LocalStorage for user preferences and profile data

UI/UX: Responsive design, mobile-first approach

Algorithms Used

  • Stream Selection: Percentage-based threshold algorithm
  • Placement Prediction: Weighted multi-factor scoring (CGPA×10 + Skills×0.3 + DSA×0.35 + Projects×2)
  • Skill Gap: Set difference comparison algorithm
  • Progress Tracking: Percentage-based completion with delta calculation
